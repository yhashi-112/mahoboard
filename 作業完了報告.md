## 🚨 重要な注意事項

### ファイル構成の確認
```
your_project/
├── app.py                          # ← 必ず置き換え
├── knowledge_base/                 # ← 必ず作成
│   └── qa_knowledge_base_all.json # ← 必ず配置
├── .env
└── requirements.txt
```

### 環境変数
- `OPENAI_API_KEY`: 既存の設定を継続使用
- `RAG_ENABLED`: デフォルトで有効（.envで変更可能）

### バックアップ
